# 矩阵基础辅助学习工具 
## 功能
- 矩阵输入和解析  
- 初等行变换和列变换（加减、倍乘、交换） 
- 行列式计算
## 使用方法
  ### 输入框的提示，示例按钮足以看懂用法了，看见可以点的东西试着按一按，总会明白的。如果还有不会再来看这里
1. 矩阵录入
  1. 在第一个输入框输入矩阵：元素间使用逗号（全角半角都行）隔开，行间使用分号（全角半角都行）；
  2. 输入完整矩阵后，点击“录入矩阵”按钮，即可完成矩阵录入，页面出现解析后的矩阵以及对应的行列索引按钮
2. 初等变换
  1. 你可以在输入框输入内容，在“选择操作”按钮选择初等变换方法（加减、倍乘、交换），程序会根据你的输入和选择合成历史记录、执行初等变换
  2. 第一个输入框，输入要进行变换的【目标行或列】
  3. “选择操作”按钮，选择初等变换方法（加减、倍乘、交换）
  4. 第二个输入框，输入要进行变换的参数，如：若要进行加减操作，输入【行列索引】或【系数*行列索引】作为参数；若要进行交换操作，输入【行列索引】作为参数；若要进行加减操作，输入【数字】作为参数
  5. 快捷符号栏：点击运算符号，可以在“选择操作”快速输入或更改初等变换方法（加减、倍乘、交换）
  6. 解析后的矩阵旁有行列索引按钮，点击这些按钮可以快速在初等变换的两个输入框输入内容
7. 计算行列式的值：计算行列式的值，只有方阵可以计算
## 优势
- 拥有压缩版工具，性能高，占用空间小
- 完整的错误检测机制
- 支持初等变换的记录、撤销和重做功能
- 响应性调整，适配电脑、手机
- 简洁舒适的界面
## 限制
- 只能输入整数和小数，无法识别未知数，分数
- 仅支持有限维度的矩阵和有限次的初等变换
- 只进不出，没有复制功能
